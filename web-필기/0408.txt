 +front-end,화면 구현의 구성요소

 -HTML : 웹 화면의 구조를 구성하는 언어

 -CSS:화면의 스타일을 지정하는 언어

 -Javascript: 화면의 동작을 정의하는 언어

 +HTML
 -HyperText Markup Language
	HyperText:하이퍼링크가 적용된 문서(다른 문서와 연결된 문서)
	Markup Language:태그를 사용하여 문서나 데이터의 구조를 나타내는 언어

 -웹페이지의 정적인 구조를 표현하는 언어
 -웹페이지의 골격을 개발하는 언어이다.

 -웹UI의 레이아웃을 구성한다
 -테그를 이용하여 개발한다

 +태그,tag

 -HTML 레퍼런스
 - <여는태그>와 </닫는테그> 두개를 한 쌍으로 구성된다.

 -HTML문서의 기본테그:<HTML>,<HEAD>,<BODY>
 -위 세가지 태그는 html문서에 반드시 포함되어있어야 한다.

 -기본테그들의 형식
	<html>
		<head>
		<body>

		</body>
		</head>
	</html>

 **<html> 태그 : HTML문서의 최상위 태그,모든 태그를 감싸고 있음

 **<head> 태그
	HTML문서의 정보를 입력하는 태그
	HTML문서 자체의 정보나 설정값들을 입력한다

 **<body>태그
	화면에 보여질 내용을 작성하는 곳
